How to make build app?
Requirements:
  - Cordova
  - Java JDK
  - Android SDK
  - Android target
  - Gradle

Make build:
  - Move the source code to the cordova\www directory. 
  - Run the "cordova build android" command there. 
  - You should see the results in the directory platforms/android/app/build/outputs/apk.
